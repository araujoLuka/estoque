* * * Projeto de Controle de Estoque (Excel - VBA) * * *

Autor: Lucas Araujo
Discricao: Esse é um projeto realizado para empresa JC ARAUJO para controle de Estoque, utilizando o software Microsoft Excel e a linguagem de programação integrada VBA
Linguagem: VBA

Os elementos presentes nesse projeto são definidos da seguinte forma:
    - forms: contem os codigos-fonte dos formularios utilizados no projeto
    - plans: contem as planilhas exportadas do projeto
    - scripts: contem os codigos-fonte dos algoritmos principais do projeto
    - resources: contem materiais necessarios ao projeto
    - notes: contem notas de atualizações e notas do andamento atual do projeto
    - others: contem os demais arquivos que são utilizados para desenvolvimento

* Como executar
    - Para visualizar o projeto é necessario possuir o sistema Windows e o Microsoft Excel
    - Baixe o arquivo "Estoque.xlsm" e a pasta "resources"
    - Caso prefira, baixe todo o repositorio, armazene em Documentos e crie um atalho da planilha para area de trabalho para organizar
    - Abra a planilha e habilite as macros e o conteudo
    - Para acessar ao sistema:  usuario: admin - senha: admin
